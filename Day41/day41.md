## What the App Covers
1. **Module 1 — Foundations:** Understanding Correlation (Pearson's r, scatter plot shapes, correlation vs. causation, interactive click-to-plot exercise)
2. **Module 2 — Simple Linear Regression:** least-squares line fitting, residuals, R², slope/intercept slider exercise
3. **Module 3 — Multiple Regression & Model Evaluation:** multiple predictors, adjusted R², LINE assumptions, residual-plot diagnosis exercise
4. **Module 4 — Practical Application & Mastery:** non-linear relationships, Anscombe's Quartet, Simpson's Paradox, case-study exercise
5. **Final Practical Challenge:** a 10-point study-hours-vs-exam-score dataset with a 4-part interpretation challenge
6. **Cheat sheet, printable summary notes, and a Continue Learning section** (books, docs, papers, communities, practice platforms, search keywords, follow-up AI prompts)

Each module includes detailed explanations, topic-specific examples, analogies, SVG diagrams, comparison tables, a hands-on interactive exercise, common misconceptions, key takeaways, and a 4-question auto-scored quiz with instant feedback and explanations. Progress, XP, and quiz scores persist locally via `localStorage`, and modules unlock sequentially as each quiz is completed.
