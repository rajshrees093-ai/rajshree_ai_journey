# Day 11 – ATS Resume Optimizer & Resume Generator

## Objective

Learn how to analyze and optimize a resume for Applicant Tracking Systems (ATS) using Claude AI.

## Tools Used

* Claude AI
* Resume (PDF/DOCX)
* Job Description (JD)
* GitHub
* VS Code

## Process Followed

1. Opened Claude AI and set the effort level to Low.
2. Uploaded my resume.
3. Selected a relevant Job Description for my target role.
4. Used the ATS Resume Optimizer prompt.
5. Generated an ATS analysis report.
6. Reviewed the ATS Match Score and Gap Analysis.
7. Identified missing keywords and improvement areas.
8. Generated an ATS-friendly optimized resume.
9. Saved the optimized resume.
10. Captured screenshots of the ATS analysis and optimized resume.
11. Added all deliverables to the Day11 folder.
12. Committed and pushed the changes to GitHub.

## Learnings

* ATS systems heavily rely on keywords from job descriptions.
* Resume formatting significantly affects ATS readability.
* Tailoring a resume for each job application improves the chances of shortlisting.
* Quantifiable achievements and action-oriented language strengthen a resume.
* AI tools can speed up resume optimization while maintaining accuracy.
