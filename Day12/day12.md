# Day 12 - Job Search & Personal Branding Toolkit

## Objective
Build a complete career growth toolkit using Claude AI.

## Target Role
- AI/ML Engineer
- Software Engineer

## Target Companies
- Amazon
- Google
- Microsoft
- Meta
- Apple
- Deloitte

## Deliverables
- ATS-Friendly Cover Letter
- Recruiter Outreach Email
- Hiring Manager Email
- LinkedIn Connection Request
- Referral Request Message
- Follow-Up Email
- Professional Introduction
- Skill Gap Analysis
- Personal Branding Summary

## Key Learnings
1. Personal branding matters as much as technical skills.
2. ATS optimization improves recruiter visibility.
3. Networking and referrals increase interview opportunities.
4. Identifying skill gaps helps create a focused learning roadmap.
